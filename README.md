# mini-chat-system-java
A console-based Mini Chat System in Java demonstrating arrays, recursion, file handling, exception handling, and basic chatbot functionality.
-------✨ Features
📩 Send and store messages
🤖 Built-in chatbot with automatic responses
🔍 Search messages using keywords
🗑️ Delete messages
📜 View complete chat history
🧹 Clear the chat history
💾 Save conversations to a text file
🕒 Display timestamps for messages
📋 Menu-driven console interface
--------🛠️ Technologies Used
Java | Arrays | Methods | Recursion | File Handling | Exception Handling | LocalTime & DateTime API
--------📂 Project Structure
--MiniChatSystem.java
--chatHistory.txt
--README.md
----------🚀 How to Run
-Clone this repository.
-Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
-Compile and run MiniChatSystem.java.
-Follow the on-screen menu to use the chat system.
