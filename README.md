# mini-chat-system-java
A console-based Mini Chat System in Java demonstrating arrays, recursion, file handling, exception handling, and basic chatbot functionality.
<br>
-------✨ Features
<br>
📩 Send and store messages
<br>
🤖 Built-in chatbot with automatic responses<br>
🔍 Search messages using keywords<br>
🗑️ Delete messages<br>
📜 View complete chat history<br>
🧹 Clear the chat history<br>
💾 Save conversations to a text file<br>
🕒 Display timestamps for messages<br>
📋 Menu-driven console interface<br>
<br>
--------🛠️ Technologies Used<br>
Java | Arrays | Methods | Recursion | File Handling | Exception Handling | LocalTime & DateTime API
<br>
--------📂 Project Structure<br>
--MiniChatSystem.java<br>
--chatHistory.txt<br>
--README.md<br>
<br>
----------🚀 How to Run
<br>
-Clone this repository.<br>
-Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).<br>
-Compile and run MiniChatSystem.java.<br>
-Follow the on-screen menu to use the chat system.
